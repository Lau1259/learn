CSS Grid Demo

CSS Styles:

.grid {

display: grid

grid-template-columns: repeat( , 1fr)

grid-template-rows: repeat( , 1fr)

align- :

justify- :

width:

height:

}

.grid-item {

width:

height:

}

Extra information:

The grid in the example has the class grid and uses the above style rules to modify it's layout. I am using repeat in both the column and rows because all columns and rows are to be the same size, though this can be changed by explicitly setting a size (height or width) for each column/row.

The size used in the grid above is achieved by using 1fr on the repeat statement for both the columns and rows. This will automatically set an equal size for each column and row.

Note that, if you have less rows than necessary, the grid will automatically resize any rows after the specified amount to fit the content inside of it.
