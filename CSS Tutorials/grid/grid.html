CSS Grid Tutorial

CSS Styles:

.tutorial {

display: grid

repeat( , 1fr)

repeat( , 1fr)

}

Extra information:

The grid in the example has the class tutorial and uses the above style rules to modify it's layout. I am using repeat in both the column and rows because all columns and rows are to be the same size.

The size is achieved by using 1fr on the repeat statement for both the columns and rows. This will automatically set an equal size for each column and row.

Note that, if you have less rows than necessary, the grid will automatically resize any rows after the specified amount to fit the content inside of it.
