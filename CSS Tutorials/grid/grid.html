CSS Grid Tutorial

CSS Styles:

.tutorial {

display: grid

repeat( , 1fr) repeat( , 1fr)

}

Extra information:

The grid in the example has the class tutorial and uses the above style rules to modify it's layout. I am using repeat in both the column and rows because all columns and rows are to be the same size. That is 1fr which basically tells the browser to auto-size them to fit. Note that, if you have less rows than necessary, the grid will auto size any rows under it to fit the content inside of it.
